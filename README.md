<p align="center">

</p>

<hr>

<h2>🚀 Project Overview</h2>

<p>
Talking Rabbitt is a conversational analytics prototype that allows users to interact with their business data using natural language.
</p>

<p>
Instead of manually filtering Excel sheets or navigating complex dashboards, users can simply ask questions about their data and receive instant insights with automated visualizations.
</p>

<p>
This project was built as part of the <b>AI Product Manager Challenge by Rabbitt AI</b>.
</p>

<hr>

<h2>💡 Problem</h2>

<p>
Traditional Business Intelligence tools require multiple steps to extract insights from data. Business leaders often rely on dashboards or analysts to answer simple questions.
</p>

<p>Example question:</p>

<pre>
Which region generated the highest revenue in Q1?
</pre>

<p>
Finding this answer often involves:
</p>

<ul>
<li>Opening dashboards</li>
<li>Applying filters</li>
<li>Running reports</li>
<li>Manual data analysis</li>
</ul>

<p>
This process can take <b>10–15 minutes for a single insight</b>.
</p>

<hr>

<h2>✨ Solution – Talking Rabbitt</h2>

<p>
Talking Rabbitt introduces a <b>Conversational Intelligence Layer</b> on top of business data.
</p>

<p>Users simply ask questions like:</p>

<pre>
Which region had the highest revenue?
</pre>

<p>The system instantly provides:</p>

<ul>
<li>Clear text insights</li>
<li>Automated visualizations</li>
<li>Data-driven answers</li>
</ul>

<p>
This reduces analysis time from <b>10 minutes to just a few seconds</b>.
</p>

<hr>

<h2>⚡ Key Features</h2>

<h3>💬 Conversational Query Layer</h3>
<p>Ask questions about your data in natural language.</p>

<h3>📊 Automated Visualization</h3>
<p>Charts such as bar graphs and comparisons are generated automatically.</p>

<h3>⚡ Instant Insights</h3>
<p>Business insights are delivered instantly without manual analysis.</p>

<hr>

<h2>🧠 MVP Magic Moment</h2>

<pre>
Upload CSV → Ask Question → Get Instant Insight + Visualization
</pre>

<p>
This demonstrates the core value of Talking Rabbitt: turning complex analytics into simple conversations.
</p>

<h2>⚙️ Installation</h2>

<p>Clone the repository:</p>

<pre>
git clone https://github.com/your-username/talking-rabbitt-mvp.git
</pre>

<p>Navigate into the project:</p>

<pre>
cd talking-rabbitt-mvp
</pre>

<p>Install dependencies:</p>

<pre>
pip install -r requirements.txt
</pre>

<p>Run the application:</p>

<pre>
streamlit run app.py
</pre>

<hr>

<h2>🔑 Environment Variables</h2>

<p>Create a <b>.env</b> file:</p>

<pre>
OPENAI_API_KEY=your_api_key_here
</pre>

<p><b>Important:</b> Never upload your .env file to GitHub.</p>
